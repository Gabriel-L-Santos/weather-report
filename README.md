Este é Weather Report, site de previsão do tempo usando HTML, CSS, Javascript e o consumo de diversas API's. A principal delas é a `OpenWeather`, a API usada para buscar e exibir as informações da previsão do tempo de qualquer cidade do mundo, a `FlagsAPI`, usada para buscar e exibir as bandeiras dos países ao qual as cidades pesquisadas pertencem, e a `Unsplash`, usado para buscar e exibir imagens relacionadas às cidades pesquisadas ao fundo da página. Para começar, basta pesquisar qualquer cidade e clicar no ícone de pesquisa ou pressionar o ENTER. Por fim, os ícones abaixo do meu nome levam para os meus perfis no Linkedin e GitHub.
